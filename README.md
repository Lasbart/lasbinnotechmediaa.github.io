## ğŸ› Troubleshooting

### Links Not Working?
- Check that all file names are spelled correctly
- File names are case-sensitive on GitHub
- Ensure paths start with `./` or no prefix

### CSS Not Loading?
- Check the path: `<link rel="stylesheet" href="css/styles.css">`
- Ensure the `css` folder exists
- Check file name spelling

### JavaScript Not Working?
- Check browser console for errors (F12)
- Ensure script is at the bottom of the body tag
- Check file path: `<script src="js/script.js"></script>`

### Images Not Showing?
- Check file paths are correct
- Ensure image files are uploaded
- Check file extensions (.jpg, .png, .gif)
